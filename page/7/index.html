【数组声明】
1.语法格式：
数据类型[] 数组名;
数组类型 数组名[];
例:
int[] myIntArray;
int myIntArray[];
char[] ch;
String [] strArray;
2.数组创建
语法格式一：先声明后创建
数据类型[] 数组名;
数组名 = new 数据类型[数组长度];
例：
int[] arr;
arr = new int[10];
创建了一个长度为10的整型数组
语法格式二：声明的同时创建数组
数据类型[] 数组名 = new 数据类型[数组长度];
int[] arr = new int[10];
创建长度为10的整型数组arr
注意：数组长度必须指定

变量的概念与数据类型

Posted on 2019-01-17 Views: Valine:

1.什么是变量

数据临时存储的位置

变量的三个元素：变量类型、变量名和变量值。

命名规则：
1、满足标识符命名规则
2、符合驼峰法命名规范
3、尽量简单，做到见名知意
4、变量名的长度没有限制

驼峰法命名：
单个单词，全部小写
多个单词，第一个单词首字母小写，后面的单词首字母大写。

类命名规则：
Pascal原则：组成类名的单词的首字母都要大写。
