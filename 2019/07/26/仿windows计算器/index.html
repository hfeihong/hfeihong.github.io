*名称：仿windows计算器
*功能：

  • 1、实现单击按钮录入数字
  • 2、实现基础四则运算功能，并添加必要的异常处理，例如，除数为零
  • 3、实现小数点功能并添加异常处理：小数点只能出现一次
  • 4、实现正负号功能
  • 5、实现退位功能，已经是最后一位时，显示框显示为零
  • 6、清屏功能
  • 使用的知识点:
  • 1、利用大量的自定义函数实现业务逻辑
  • 2、灵活运用事件及事件处理
  • 3、培养异常处理的编程方法
  • 4、培养并实践利用不同思路实现编程
  • 综合练习的目的：
  • 1、将css、html和js有效的进行技术组合，实现业务功能
  • 2、锻炼和培养编程思想，解决问题的能力和方法
  • 3、锻炼和培养利用多种编程思路，完成预先设定的目标
