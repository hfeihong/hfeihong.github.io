数据类型转换

Posted on 2019-01-18

类型转换
自动：隐式转换
基本规律：范围小→范围大
强制：会引起数据丢失
如图

实线：无信息丢失数据类型
虚线：转换时可能会数据丢失

每一个基本数据类型都是有范围的。你可以在控制台输出查询，代码如下：System.out.println(Long.MAX_VALUE);//最大值查询
System.out.println(Long.MIN_VALUE);//最小值查询
如果想查其他类型的范围，把对应的Long型换成你想查询的类型即可。

强制数据类型转换格式： （数据类型）数值

例：

double d=123.4;

float f=(float)d;

常量
在变量前加上final就是常量
例: final int N=333;
常量值不可改变
变量值可以改变
常量命名规则
一般用大写表示，一个单词使用全大写 例 final double PI=3.14; 多个单词则全大写，每个单词中间用_连接 例 final double MIN_VALUE=0;
